Coaching Management System API <br>
A Coaching Management System REST API built using ASP.NET Core Web API.<br>
This project manages Students, Courses, Batches, Fees, Attendance, and provides Dashboard insights.<br>
🚀 Features<br>
👨‍🎓 Student Management<br>
📚 Course & Batch Management<br>
💰 Fees Management (Partial / Paid / Pending)<br>
🧾 Auto Fee Calculation (Paid, Remaining, Status)<br>
🗑 Soft Delete<br>
📊 Dashboard Summary<br>
🔍 Pagination & Search<br>
🔐 Clean Architecture (Service Layer Pattern)<br>

RESTful API Design<br>
Entity Relationships (One-to-Many)<br>
DTO Pattern<br>
